# Guitar Diatonic Chord Finder

Select a major key, view its seven diatonic triads, then click a chord to see chord tones, fretboard positions, and playable guitar voicing candidates.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm test
```

## Deploy

Use Vercel with:

```text
Build command: npm run build
Output directory: dist
```
